LIS Form Ordering

Toolkit version: Elements v1.0.22954

Description

Users who want to apply for the NHS Low income scheme can order paper forms to be delivered using LIS Form Ordering.

Notes

LIS form ordering is hosted on SVN and required a Jboss server to run locally.

When deploying front end changes to LIS form ordering remember to get the extract tested. To make sure any updates to the system don't cause the extract to fail.
